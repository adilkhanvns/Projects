Angular js
  • Angular js is a java script framework to build an application that run on web browser using html
  • Two basic requirement to add angular to web page is
  • 1 -Script tag pointing to angular.js, angular has no dependency on other libraries.
  • 2-add ng –app some where in your application , it’s a special directive that angular looks for in html
  • And if it is find angular will bootstrap itself , so it initializes it self and jumps into action for managing the page .
  • There should be only one ng-app directive per page
